Web browser : is an interface to connect and get any information from webservers

through which user can send request and get response 

on the web to get any information we use browsers 

when user sends query , then query engine searches
using keywords, headings (info under heading tags) or title of the web page and returns matching one

1.You type a URL (e.g., www.google.com) in a browser 2.like Google Chrome
The browser sends a request using HTTP/HTTPS
The request goes to a server (computer on the internet)
That web server receives your query and processes it
Server sends back the response (webpage)
